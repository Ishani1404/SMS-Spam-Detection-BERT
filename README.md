# SMS-Spam-Detection-BERT
## Problem Statement
Detect whether an SMS message is Spam or Ham using Deep Learning Model BERT.
## Dataset
https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset
Labels: Ham (0), Spam (1)
## Model Used
Pretrained BERT (bert-base-multilingual-cased)
## test accuracy achived
99.4%
